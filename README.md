# Maryland Crime Data Analysis

This project analyzes Maryland crime data across jurisdictions using Python. The analysis explores crime rates, correlations between crime categories, and crime trends over time.

## Project Overview

The goal of this project was to investigate patterns in Maryland crime data and answer questions such as:

- Which Maryland jurisdictions have the lowest crime rates?
- How strongly are different crime categories correlated?
- Did average total crime per jurisdiction change significantly between 2000 and 2019?

## Key Analyses

- Calculated crime rate per 1,000 residents to compare jurisdictions fairly
- Visualized the top 15 safest Maryland jurisdictions
- Created a correlation heatmap of major crime categories
- Compared average total crime per jurisdiction in 2000 vs. 2019
- Performed hypothesis testing using two-sample t-tests

## Main Findings

The analysis found that normalizing crime by population is important for fair comparison across jurisdictions. The project also found strong correlations between many crime categories and a statistically significant decline in average total crime per jurisdiction between 2000 and 2019.
